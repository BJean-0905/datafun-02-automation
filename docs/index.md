## Datafun 02 Automation

## Dataset

Predefined datasets provided in the project. Pet types: cat, dog, fish. Favorites: cat, dog, fish.
Quarterly data covers quarters one through four.
Periodic files 1-3, added a periodic file 4. Added additional data to the provided data to include pet
name and age. Added to the favorite files: color and species.

## Signals

Loops are designed to run the report periodically while pausing for 1 second between running each file.

## Experiments

Added an artifacts folder to distinguish between preset data and actual output after running the
script. Adjusted the number of periodic files to be 4 instead of 3 and confirmed a fourth file
appears after running the script. Attempted to add the additional data to the final output in
artifacts unsuccessfully.
Column designations and a dataframe will need to be structured in the code to pull in
the additional information.

## Interpretation

Loops can be beneficial for tracking overtime, automating oversight of
workflows, operations, and systems. Utilizing actual content inside the
files would have been more helpful to see output represented in each file
in a way that connected the output more congruently. For example, a
dataframe with columns could pull in the information provided. Adding
an artifacts folder was really helpful in delineating the before data
and after data.
